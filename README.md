## american-regiment

> vue-cli2.0仿美团移动端项目

#### 项目运行
1. 项目初始化(前提是先切换到项目文件所在的盘符）
```
npm install
```

2. 运行项目
```
npm run dev
```

3. 项目打包
```
npm run build
```
